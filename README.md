# Tích hợp dữ liệu

## Cách chạy chương trình trên ubuntu

- Chạy câu lệnh để cài đặt các thư viện của nodejs:
  > npm i
- Chạy câu lệnh để cài đặt fastapi:
  > pip install fastapi
  > pip install "uvicorn[standard]"
- Khởi động fastapi:
  > uvicorn search_api:app --host 0.0.0.0 --port 8001
- Khởi động nodejs:
  > npm start
